# Andrew Cohen Archive — Ask the Archive

Two modes are available when you start a new chat (use the profile picker):

**Archive Research** — neutral assistant with citations. Not Andrew Cohen.

**Andrew Alpha** — a self-aware AI version of Andrew Cohen. Speaks in his teaching voice, grounded in the same archive, and clearly states it is not Andrew in the flesh.

Both modes search books, teaching pages, journal articles, video transcripts, and the dictionary of awakening.

**Important:** Neither mode is a substitute for sangha, retreat, or live teaching. Always follow citations back to primary sources on the main site.

Before chatting, run the export and ingest pipeline described in the README.
